## 摘要
在线策略自蒸馏是一种有前景的密集监督来源，它通过语言模型利用特权上下文来监督自身的生成，用于稀疏奖励强化学习。实际上，它可以实例化为一个辅助的全词汇学生到教师反向KL散度损失。因此，我们提出了SDPG，一个自蒸馏策略梯度框架，它结合了分组相对验证器优势、归一化标准差、精确的全词汇在线策略自蒸馏以及参考策略KL正则化。实验上，SDPG在RLVR和自蒸馏基线上提高了稳定性和性能。代码可在https://github.com/lauyikfung/SDPG获取。

## Abstract
On-policy self-distillation, where a language model conditions on privileged context to supervise its own generations, is a promising source of dense supervision for sparse-reward reinforcement learning. Actually, it can be instantiated as an auxiliary full-vocabulary student-to-teacher reverse Kullback-Leibler divergence loss. We therefore propose SDPG, a self-distilled policy-gradient framework that combines group-relative verifier advantages with normalized standard deviation, exact full-vocabulary on-policy self-distillation, as well as reference-policy KL regularization. Empirically, SDPG improves stability and performance over RLVR and self-distillation baselines. The code is available at https://github.com/lauyikfung/SDPG.

## 1. 核心问题与整体含义（研究动机与背景）

- **研究动机**：在大型语言模型的强化学习后训练中，基于可验证奖励的RLVR方法（如GRPO）面临两个核心挑战：① 序列级奖励稀疏，无法为每个token提供有效信用分配；② 负优势（negative advantages）在训练初期会破坏模型稳定性。已有的on-policy蒸馏方法虽然能提供密集token级信号，但通常需要额外的、更大的教师模型，带来显存负担且异构教师信号可能损害训练平滑性。
- **整体目标**：提出一种“自蒸馏”策略（即教师与学生为同一模型，仅在是否拥有特权上下文条件下不同），在保留二元验证器奖励的同时，利用全词汇反向KL散度提供密集、同质的逐token监督，并通过参考策略KL正则化增强稳定性。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将全词汇在线策略自蒸馏（full-vocabulary on-policy self-distillation）与RLVR中的分组相对优势（group-relative advantage）相结合，形成三部分损失函数：① 奖励驱动的策略梯度损失（基于REINFORCE）；② 门控全词汇蒸馏损失（学生-教师反向KL散度）；③ 参考策略KL正则化（采用未归一化KL，即UKL）。
- **关键技术细节**：
  - 模型同时作为学生（无特权上下文）和教师（有特权上下文，如正确答案、解法提示）。对每个采样前缀，计算学生分布 \(p_t\) 和教师分布 \(q_t\)，蒸馏损失为 \(\mathrm{D}_{\mathrm{KL}}(p_t \parallel \text{SG}[q_t])\)。
  - **命题3.1**：该反向KL蒸馏损失的梯度等价于一个带中心化log-ratio优势的detached-sampling策略梯度更新。
  - **门控机制**：仅对优势为正（验证器认可的）的轨迹应用蒸馏损失，避免在错误轨迹上强化教师信号。
  - **β调度器**：蒸馏系数采用“预热-衰减”调度，前期逐渐增加以稳定训练，后期衰减以减少对特权信息的依赖，鼓励探索。
  - 参考策略KL正则化使用未归一化KL（UKL），避免归一化KL中的偏差，提供两种变体：SDPG-UFKL（前向UKL）和SDPG-URKL（反向UKL）。
- **算法流程**：每个训练步，对每个提示采样G个回答，计算组内优势，然后计算每个token位置的全词汇蒸馏损失（仅当优势为正时），并与奖励损失和KL正则损失求和，更新参数。

## 6. 主要结论与发现

- SDPG（两种变体）在所有三个基准上均显著优于GRPO和RLSD，最高准确率超过GRPO约10个百分点（AIME24上0.408 vs 0.316）。
- SDPG-UFKL（未归一化前向KL正则）在大部分指标上达到最优，且保持最高模型熵（避免模式崩塌），而RLSD出现熵崩溃。
- 训练早期SDPG即拉开与GRPO差距，奖励收敛更快、更稳定。
- 消融实验表明：蒸馏项是性能快速提升的主因，KL正则项则防止响应长度过度缩短和熵失控，两者均不可或缺。
- 在1.7B模型上，纯自蒸馏基线OPCD在训练后期出现严重崩溃（准确率暴跌至0.02），而SDPG保持稳定，验证了结合验证器奖励和门控机制的必要性。
