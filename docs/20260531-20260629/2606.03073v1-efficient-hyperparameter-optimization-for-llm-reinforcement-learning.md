## 摘要
大语言模型（LLM）的强化学习对超参数配置高度敏感，因此超参数优化（HPO）至关重要但计算成本高昂。由于模型规模巨大且训练周期资源密集，现有的多保真度HPO方法对LLM强化学习仍不够高效。本文提出联合保真度超参数优化（JF-HPO），该方法同时将模型大小和训练预算作为保真度进行自适应调整。JF-HPO的核心优势在于：（i）在每个HPO试验中，利用目标LLM的小型代理模型进行高效训练与评估；（ii）整合基于训练动态精心设计的早停策略；（iii）引入高效的检查点机制以消除冗余计算。与现有HPO方法相比，JF-HPO显著提升每个试验的计算效率（最高达14.9倍），同时在相同时间预算下达到更优或具有竞争力的预测精度。值得注意的是，与采用VeRL配方中的超参数配置相比，JF-HPO的性能提升幅度为5.8%至111.6%。

## Abstract
Reinforcement learning (RL) for large language models (LLMs) is highly sensitive to hyperparameter configurations, making hyperparameter optimization (HPO) essential yet computationally expensive. Existing multi-fidelity HPO methods remain inefficient for LLM RL due to the massive model scale and resource-intensive training cycles. In this paper, we propose Joint Fidelity Hyperparameter Optimization (JF-HPO), which simultaneously adapts both model size and training budget as fidelity. JF-HPO is empowered by: (i) it leverages a small proxy model of the target LLM for efficient training and evaluation in each HPO trial; (ii) it integrates carefully designed early-stopping strategies based on training dynamics; (iii) it introduces an efficient checkpointing mechanism to eliminate redundant computations. Compared with existing HPO methods, JF-HPO significantly improves the computational efficiency of each trial (up to 14.9 times), while achieving better or competitive predictive accuracy under the same time budget. Notably, compared with utilizing hyperparameter configurations from the VeRL Recipe, JF-HPO delivers performance improvements ranging from 5.8% to 111.6%.

- **核心问题**：大语言模型（LLM）的强化学习（RL）训练对超参数极为敏感，但现有的超参数优化（HPO）方法因模型规模巨大和训练周期过长而计算效率极低，难以实际应用。
- **研究动机**：RLHF/RLVR 等技术在 LLM 上取得了显著成功，但超参数的微小变化（如学习率、裁剪比例）会严重影响最终性能。传统 HPO 方法（如网格搜索、随机搜索、BOHB）在 LLM RL 场景下每个试验都需要训练大模型，时间成本不可接受。
- **研究意义**：将 HPO 应用于 LLM RL 以提升性能，同时大幅降低计算开销，使 HPO 在 LLM 领域变得可行和高效。

## 2. 论文提出的方法论

### 2.1 核心思想
提出 **JF-HPO（Joint Fidelity Hyperparameter Optimization）**，**同时将模型大小和训练预算作为两维保真度**进行联合优化。通过使用目标 LLM 的小型代理模型进行快速训练与评估，并结合早停和检查点机制，显著减少每个 HPO 试验的计算成本。

### 2.2 关键技术细节
- **代理模型**：从同一模型家族中选择一个规模小得多的模型（如 0.5B~1B 参数）作为目标模型（如 7B/8B/14B）的代理，用于低保真度下的超参数探索。通过秩相关性分析保证代理模型与目标模型的超参数排名一致性（Spearman 0.90，Kendall 0.80）。
- **联合保真度获取函数**：扩展贝叶斯优化中的期望改进（EI），将模型大小和训练步数作为保真度，并加入时间成本归一化：
  \[
  \alpha(\phi_t, r_t) = \frac{\mathbb{E}[f'(\theta',\phi_t,r_t) - f^*(\theta,\phi^+,r_{\max}) \mid \mathcal{D}]}{\mathbb{E}[C(\theta',\phi_t,r_t)]}
  \]
  其中 \(f'\) 为代理模型性能，\(r_t\) 为训练步数，\(C\) 为时间成本。
- **早停策略**：基于训练动态设计两个触发条件（连续 k=5 步）：
  - KL 散度增长比率超过阈值 \(\tau_1=15\%\) → 表示策略变化过快，不稳定。
  - 训练奖励下降比率超过 \(\tau_2=10\%\) 或持续为零 → 表示配置不利于学习。
- **高效检查点机制**：维护一个注册表，记录每个配置在不同保真度下的模型检查点。若某配置在更高保真度下继续训练，可直接加载之前低保真度检查点恢复训练，避免从头开始。
- **算法流程**（见 Algorithm 1）：初始化 GP 和注册表；迭代选择配置-保真度对；加载检查点；训练代理模型并实时监测早停；保存检查点；评估性能；更新观测数据集和 GP；最终返回最优配置。

2. **实用高效**：使用小代理模型 + 早停 + 检查点，实现了数量级的加速，使 LLM RL 的 HPO 变得切实可行。
