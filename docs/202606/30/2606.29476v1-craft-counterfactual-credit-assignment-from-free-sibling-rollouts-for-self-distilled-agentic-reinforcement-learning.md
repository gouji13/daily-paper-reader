## 摘要
自蒸馏智能体强化学习通过逐令牌蒸馏损失增强轨迹级奖励，其教师模型是同一策略但以特权上下文为条件。当前的主流方法使用单个标量——师生对数概率差距——来门控该损失。这一信号存在双重局限性：它是回顾性的，仅对实际执行的轨迹评分，从未考虑反事实轨迹；并且它对符号不敏感，从未在教师偏好的行动会损害轨迹时发出信号。我们提出CRAFT，一种三支柱信用分配方案，解决了上述两个局限性。支柱一，反事实令牌重要性，重用GRPO已经采样的G-1个兄弟轨迹，并通过对数概率差距对其进行重要性加权，以形成组级反事实优势变化的自归一化估计，该变化源于在每个步骤上提升教师偏好行动的权重；这以近乎零的额外计算量产生带符号的逐令牌信用。支柱二是一个非对称控制器，它根据门控活动的指数移动平均降低参考KL权重的同时提高蒸馏权重，反之亦然。支柱三逐令牌极化KL惩罚，根据信用的符号在模式寻找和模式覆盖更新之间切换。每个支柱都有一个独立开关，当禁用时，损失和梯度在IEEE-754算术中与基线逐字节相同，因此任何测量到的增益都可归因于算法变化而非实现漂移。我们证明了估计量的一致性和方差界，提供了结构和位精确的可重复性保证，并在三个智能体环境、四个模型规模、五种端到端方法以及两个列表化的先前工作基线上评估了CRAFT。其中包括Adaptive-CRINGE，一个与CRAFT共享支柱二的比较对象，以隔离反事实贡献。

## Abstract
Self-distilled agentic reinforcement learning augments trajectory-level reward with a token-level distillation loss, using as its teacher the same policy conditioned on privileged context. The prevailing recipe gates this loss by a single scalar, the teacher-student log-probability gap. This signal is doubly limited: it is retrospective, scoring only the realised rollout and never the counterfactual ones, and it is sign-blind, never signalling when a teacher-preferred action would have harmed the trajectory. We introduce CRAFT, a three-pillar credit-assignment scheme that addresses both limitations. Pillar 1, Counterfactual Token Importance, reuses the G-1 sibling rollouts that GRPO already samples and importance-weights them by the log-probability gap to form a self-normalised estimate of the group-level counterfactual change in advantage from up-weighting teacher-preferred actions at each step; this yields a signed per-token credit at near-zero extra compute. Pillar 2 is an asymmetric controller that raises the distillation weight as it lowers the reference-KL weight along an exponential moving average of gate activity, and conversely. Pillar 3 polarises the KL penalty token by token, switching between a mode-seeking and a mode-covering update according to the sign of the credit. Each pillar has an independent switch that, when disabled, renders the loss and gradient byte-identical to the baseline in IEEE-754 arithmetic, so any measured gain is attributable to algorithmic change rather than implementation drift. We prove the estimator's consistency and a variance bound, give structural and bit-exact reproducibility guarantees, and evaluate CRAFT across three agentic environments, four model scales, and five end-to-end methods, plus two tabulated prior-work baselines. Among these is Adaptive-CRINGE, a comparator sharing Pillar 2 with CRAFT, isolating the counterfactual contribution.

## 1. 论文的核心问题与整体含义（研究动机和背景）

当前自蒸馏智能体强化学习（Self-Distilled Agentic RL，如SDAR）使用教师策略（同一策略在特权上下文下）指导学生策略，其中关键机制是逐令牌的蒸馏损失。该损失通过单个标量——师生对数概率差距 Δ_t——来门控。作者指出这一信号存在双重局限性：

- **回顾性（Retrospective）**：仅对实际执行的轨迹评分，从未考虑反事实轨迹（如果代理在步骤 t 采取教师偏好动作会发生什么）。
- **符号盲（Sign-blind）**：门控 σ(βΔ_t) 单调递增，从未在教师偏好动作实际会损害轨迹时发出“不要跟随”的信号。

因此，当前方法将两个本质上不同的信号（回顾性 vs. 反事实）混为一个标量，导致信用分配不准确。论文的核心动机是：**在自蒸馏智能体RL中，正确的逐令牌信用信号应该是“如果代理在步骤t从教师采样，序列优势的反事实变化”**，并且GRPO已经免费提供了估算该反事实所需的采样预算（G-1个兄弟轨迹）。

## 2. 论文提出的方法论：核心思想、关键技术细节

CRAFT（Counterfactual Credit Assignment from Free Sibling Rollouts）由三个支柱构成，组成整体损失函数：

`L_CRAFT(θ) = L_PG(θ) + μ(t) L_KL.pol(θ) + λ(t) L_CTI(θ)`

其中 L_PG 是未修改的GRPO策略梯度损失。

### 2.1 支柱一：反事实令牌重要性（Counterfactual Token Importance, CTI）

**核心思想**：利用同一GRPO组中已采样的 G-1 个兄弟轨迹，通过重要性加权估算“如果提升教师偏好动作权重，组级优势的平均变化”，减去当前轨迹的实际优势，得到反事实信用。

**具体流程**：
1. 每个令牌计算师生对数概率差距 Δ_t = stopgrad[log π_T - log π_θ]。
2. 对每个轨迹 i，用其余 G-1 个兄弟的 Δ 计算 softmax 权重：`w_j ∝ exp(Δ_j / τ)`，默认 τ=1。
3. 计算组级反事实优势差：`dCTI_t(i) = Σ_{j≠i} w_j A(j) - A(i)`。这是自归一化重要性抽样（SNIS）估计量，具有一致性（Prop. 1）和 O(1/G) 方差界（Prop. 2）。
4. 通过非对称中心化sigmoid将 dCTI 映射到有符号信用 `c_t ∈ (-1,1)`，正负方向使用不同的锐度 β+, β-。
5. 最终CTI损失采用REINFORCE形式：`L_CTI(θ) = - Agg[ c_t * log π_θ(y_t | s_t) ]`，梯度仅流过 log π_θ，c_t 完全分离。

**关键优势**：不增加模型推理（已采样兄弟轨迹），仅需组级softmax和加权求和，边际计算成本几乎为零。

### 2.2 支柱二：相感知自适应控制器（Phase-Aware Adaptive Controller）

**核心思想**：蒸馏过程经历活跃期（高ρ_gate，教师信号未吸收）、稳定期、饱和期。静态权重会导致过蒸馏或欠蒸馏。控制器根据门控活动比率 ρ_gate（|c_t| > 阈值的令牌比例）动态调整两个系数：

- λ(t) = λ_min + (λ_max - λ_min) * ρ_EMA_gate(t)  （CTI损失权重，随活跃度升高）
- μ(t) = μ_min + (μ_max - μ_min) * (1 - ρ_EMA_gate(t)) （参考KL权重，随活跃度降低）

ρ_EMA_gate 通过指数移动平均更新，前 n_warmup 步保持中间值。两个系数反向移动，实现蒸馏吞吐与探索预算之间的帕累托权衡。

### 2.3 支柱三：方向极化KL惩罚（Direction-Polarised KL）

**核心思想**：先前的KL惩罚对所有令牌采用同一形式（k3）。有符号信用使更好的选择成为可能：当 c_t > 0（跟随教师有帮助）时使用模式寻找（reverse KL，拉向π_ref）；当 c_t < 0（跟随教师有害）时使用模式覆盖（forward KL符号翻转，推离π_ref）；|c_t| ≈ 0 时使用标准KL。

具体规则：
- `c_t > θ_KL`：rev(t) = log π_θ - log π_ref （最小化此值，拉向参考，模式寻找）
- `c_t < -θ_KL`：fwd(t) = - (log π_θ - log π_ref) （符号翻转，推开，模式覆盖）
- `|c_t| ≤ θ_KL`：std(t) = 标准KL惩罚（k3）

负分支是前向KL启发式更新规则（非无偏估计量），但梯度恒定 ±1，保证训练稳定性。

### 2.4 可重复性保障（P7）

每个支柱都有一个独立主开关，关闭时（`enable=False`, `phase_aware=False`, `polarized_kl=False`）损失和梯度与基线（SDAR）在IEEE-754算术中逐字节完全相同，确保任何测量增益完全归因于算法变化而非实现漂移。

## 6. 论文的主要结论与发现

1. **CRAFT-Full在所有（环境，模型）单元上均为最佳**（表2），超越所有基线，包括最硬的Adaptive-CRINGE。
2. **CTI（P1）贡献主要增益**（A1相比SDAR提升约+1.5~+2.0个百分点），P2和P3提供微小增量（≤0.5）。

4. **大小模型缩放规律**：小模型上增益最大（Qwen3-1.7B达+5.5），大模型上收缩但保持正值（~1.6-2.1），符合预期（大模型在更多令牌上师生一致，ρ_gate较小）。
5. **OOD泛化更好**：CRAFT-Full在三个环境上的分布内→OOD差距最小（ALFWorld 6.2 vs SDAR 9.5）；**随机检索下降更缓和**（78.5 vs SDAR 72.0）。
6. **伪造检验通过**：KL方向交换（A7）导致性能低于SDAR基线，验证了P3的设计方向正确；禁用CTI（A6）恢复SDAR基线附近，确认CTI的核心作用。

## 7. 优点

- **近乎零额外计算**：重用了GRPO已采样的兄弟轨迹，仅需组级softmax和加权求和，边际成本可忽略。
- **理论严谨**：CTI估计量一致且具有O(1/G)方差界（Prop. 1, 2）；各命题有完整证明；P7提供位精确可再现性。
- **模块化设计**：三个支柱各独立主开关，便于消融和实验归因；自适应控制器分离了蒸馏权重和KL权重两个耦合变量。
- **信号丰富**：有符号信用 c_t ∈ (-1,1) 同时编码了sign(A)和sign(Δ)，覆盖四象限（表5），比单门控更细粒度。
- **严格的实验比较**：Adaptive-CRINGE共享P2的设计巧妙隔离了P1+P3贡献；伪造检验验证设计选择；字节相等性保证实现公平。
- **全面基准**：覆盖四种模型规模、三个不同任务领域、五种端到端方法加两种先前工作基线。

## 8. 不足与局限

- **L1 - 组轨迹需求**：P1需要G≥2；非组骨干（如PPO单轨迹）需要通过合成兄弟来应用，作为未来工作。
- **L2 - 超参数数量**：引入9个新超参数（β+, β-, τ, θ_g, λ_min/max, μ_min/max, α, n_warmup, θ_KL），尽管提供了默认值并在所有实验中固定。
- **L3 - P3独立增益小**：极化KL本身贡献微小（约0.3-0.5%），主要是“完整性”贡献而非性能驱动。论文承认这一点。
- **L4 - 领域覆盖**：所有基准均为文本智能体；OOD分析在领域内（任务模板哈希分割）；多模态和跨领域泛化作为未来工作。
- **L5 - P3负分支非无偏KL估计量**：是前向KL启发式更新规则，不是无偏估计量；重要权重替换方案具有无限方差（文中已讨论）。论文将其作为诚实更新规则，而非KL估计量。
- **潜在偏差风险**：实验仅在Qwen系列模型上进行，未测试其他架构（如LLaMA、Mistral）；超参数未针对每个环境调整，虽然公平但可能隐藏调优上限。
- **缺少过程奖励对比**：未与基于过程奖励模型（PRM）的方法（如Math-Shepherd）比较，尽管声明是互补的。
