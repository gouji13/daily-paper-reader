## 摘要
在线策略蒸馏通过教师模型提供的密集、令牌级监督提升了标准强化学习的学习效率。在OPD的标准KL目标中，令牌级损失被均匀平均，意味着所有令牌权重相等。然而，我们发现并非所有令牌都是平等的：随着学生轨迹变长，它们与教师分布的偏离增大，导致后期位置的监督质量下降。因此，仅使用前30%令牌的OPD可以取得与使用全部令牌相当的表现，而仅使用后30%令牌的OPD几乎学不到任何东西。本文通过约束优化的视角，对这一现象提供了原则性的理解。基于这些见解，我们推导出重要性加权在线策略蒸馏（IW-OPD），其中分配给每个令牌的权重取决于学生与教师分布的累积差异，自然地提高早期令牌的权重，降低偏差较大的后期令牌的权重。我们表明，IW-OPD比OPD收敛更快，学习效率更高，并且在同规模和跨规模设置中均达到比标准OPD更好的最终性能，在AIME-2025上性能提升高达6.9个点。

## Abstract
On-Policy Distillation (OPD) improves the learning efficiency of standard reinforcement learning through dense, token-level supervision from teachers. In the standard KL objective of OPD, token-level losses are uniformly averaged, implying equal weights for all tokens. However, we discover that not all tokens are created equal: as student rollouts grow longer, they deviate further from the teacher's distribution, leading to degraded supervision quality at later positions. As a result, OPD using only the first 30% of tokens can perform comparably to using all tokens, whereas OPD using only the last 30% of tokens barely learns anything. In this work, we provide a principled understanding of this issue through the lens of constrained optimization. Based on these insights, we derive Importance-Weighted On-Policy Distillation (IW-OPD), in which the weight assigned to each token depends on the accumulated discrepancy between the student's and teacher's distributions, naturally upweighting earlier tokens and downweighting later ones with larger deviations. We show that IW-OPD converges significantly faster than OPD, with better learning efficiency, and achieves better final performance than standard OPD in both same-size and cross-scale settings, improving performance up to 6.9 points on AIME-2025.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：在在线策略蒸馏（On-Policy Distillation, OPD）中，学生模型（student）通过自己生成的轨迹（rollout）接受来自教师模型（teacher）的密集、token 级监督。标准 OPD 对所有 token 的 KL 散度损失进行**均匀平均**，但论文发现并非所有 token 都同等重要：学生轨迹越长，后期 token 与教师分布偏离越大，导致监督质量急剧下降。具体地，仅使用前 30% token 的 OPD 即可达到全量 token 的性能，而仅使用后 30% token 的 OPD 几乎无法学习。
- **研究动机**：揭示 OPD 中存在的**位置偏差**现象，并解释其根源——学生轨迹早期 token 与教师分布兼容性高，后期 token 则低。标准 OPD 均匀分配梯度预算是对有限更新资源的低效利用。
- **整体含义**：这项工作通过约束优化的视角，将 OPD 视为在有限本地更新预算下的局部投影问题，从而提出一种更高效、自适应的加权方案，提升蒸馏效率。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：从约束优化角度，将 OPD 训练建模为在当前学生分布 πθ 的局部区域内（KL 散度约束 ρ 内）寻找最接近教师分布 πT 的分布 q*。解该优化问题得到最优策略 q* ∝ πθ(y)·(πT(y)/πθ(y))^α，即按学生-教师的似然比（likelihood ratio）重新加权。由于后期 token 的似然比（前缀概率比）较低，q* 自然赋予它们更小的采样概率，从而解释位置偏差。
- **提议方法**：**重要性加权在线策略蒸馏（Importance-Weighted On-Policy Distillation, IW-OPD）**。
  - 直接优化 q* 与 πT 的 KL 散度，通过重要性采样变换为标准 OPD 形式，每个 token 的 KL 项乘以归一化、停止梯度的前缀重要性权重 ˜r_t。
  - **稳定权重估计**：直接使用理想似然比会导致数值不稳定，因此论文提出了实用替代方案：
    1. **对数缩放**：累加 token 级 OPD 优势（A_t^{OPD}）作为前缀不兼容性度量。
    2. **符号修正**：取绝对值累加（|A_t^{OPD}|），避免正负抵消。
    3. **归一化**：将累积绝对值缩放到 [0,1]。
    4. **插值**：与标准 OPD 融合，保留密集监督基线：˜r_t^{IW-OPD} = 1 + γ·(1 - (Σ_{k<t}|A_k^{OPD}|)/(Σ_{k<T}|A_k^{OPD}|))，默认 γ=0.5。
  - 最终梯度形式：A_t^{IW-OPD} = sg[˜r_t^{IW-OPD}]·A_t^{OPD}。

## 6. 论文的主要结论与发现

1. **位置偏差存在且显著**：OPD 中早期 token 贡献绝大部分学习收益；后期 token 监督低效，甚至有害。
2. **IW-OPD 提升样本效率**：在训练早期（step 10）即明显领先 OPD，且更快收敛。
3. **IW-OPD 提升最终性能**：在所有教师-学生配对中，平均超越标准 OPD，在 AIME-2025 上最高提升 6.9 点。
4. **优势随教师-学生不匹配增大而增强**：更强的教师或更小的学生（更大压缩比）时，IW-OPD 相对 OPD 的增益更大。
5. **可与其他方法正交结合**：IW-OPD 可与 ExOPD（另一种 reward 设计方法）结合，进一步提升性能（表 7）。
