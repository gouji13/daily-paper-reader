## 摘要
大型语言模型（LLM）的强化学习通常依赖信任域掩码来稳定离策略更新。主流的PPO风格方法使用采样令牌的重要性比率来满足两个准则：邻近性准则（检查策略是否偏离行为策略过远）和方向性准则（检查更新是否使其更远）。最近的工作DPPO通过用行为策略与训练策略之间的概率散度替换PPO基于比率的测试，改进了邻近性准则。然而，其方向性准则仍继承自PPO，令牌仅在采样令牌重要性比率偏离1时被掩码。我们观察到，这种基于比率的方向性准则是单样本代理，其符号可能与定义邻近性准则的散度变化不一致。因此，我们提出预测散度掩码，它询问下一个策略梯度步骤将增加还是减少信任域所使用的同一散度。对于LLM RL中使用的离散softmax策略，我们推导出这种预测的闭式解。由于生产级回滚引擎仅暴露词汇表的截断（top-K）视图，我们为此预测开发了两种轻量级top-K估计器。详细分析表明，与采样比率相比，基于散度的方向与散度的实际变化更加一致，由此产生的掩码在不同模型规模和精度设置下改善了RL训练。

## Abstract
Reinforcement learning for large language models (LLMs) typically relies on trust-region masks to stabilize off-policy updates. The dominant PPO-style approach uses the sampled-token importance ratio for two criteria: a proximity criterion, which asks whether the policy has moved too far from the behavior policy, and a direction criterion, which asks whether the update pushes it farther away. Recent work DPPO improves the proximity criterion by replacing PPO's ratio-based test with a probability divergence between the behavior and training policies. However, its direction criterion is still inherited from PPO. A token can be masked only when the sampled-token importance ratio moves away from one. We observe that this ratio-based direction criterion is a single-sample proxy that can disagree in sign with the change of the divergence that defines the proximity criterion. We therefore propose the predictive divergence mask, which asks whether the next policy-gradient step will increase or decrease the same divergence used by the trust region. For the discrete softmax policies used in LLM RL, we derive this prediction in closed form. Because production rollout engines expose only a truncated (top-K) view of the vocabulary, we develop two lightweight top-$K$ estimators for this prediction. Detailed analysis shows the divergence-based direction is better aligned with the realized change of the divergence than the sampled ratio, and the resulting masks improve RL training across model scales and precision settings.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：LLM 的强化学习（RL）后训练（如 PPO、GRPO）常用信任域掩码来稳定离策略更新。掩码由两个准则组成：  
  - **邻近性准则**：判断当前策略是否偏离行为策略过远；  
  - **方向性准则**：判断更新是否使偏离进一步扩大（向外）还是缩回（向内）。  
  PPO 将两个准则都建立在**单采样 token 的重要性比率** \(r_t\) 之上，但该比率是散度（如 TV 距离）的粗糙单样本估计。  
- **前期工作（DPPO）**：将邻近性准则改进为直接使用**分布散度**（如 top-K KL），使信任域更准确。但**方向性准则仍沿用 PPO 的比率符号**（\(\text{sign}(\hat{A}_t(r_t-1))\)），造成**不一致**：散度测量的是全分布的变化，而方向性准则只观测单个采样 token 的比率变化，两者符号可能相反。  
- **核心问题**：当信任域用散度定义时，方向性准则也应基于该散度的变化来判定，而非基于单样本比率。  
- **整体意义**：本文提出**预测散度掩码（Predictive Divergence Mask）**，让方向性准则直接预测下一个梯度步骤将增加还是减少同一散度，从而消除不一致，提升 RL 训练的稳定性和有效性。

## 2. 论文提出的方法论

### 核心思想
- 将方向性准则从“比率是否远离 1”重构为“散度是否增大”。具体而言，对于当前 token，计算散度 \(D_t = D_{\text{KL}}(\mu \| \pi)\) 沿策略梯度方向的**一阶导数 \(\dot{D}_t\)**，其符号指示散度趋势；若 \(\text{sign}(\hat{A}_t \dot{D}_t) > 0\) 且散度已超阈值，则掩码该 token 梯度。

### 关键技术细节
1. **闭式解推导**（Proposition 1）：对于 softmax 策略，沿单位梯度方向（\(v_i = \mathbf{1}[i=k] - \pi_i\)）的 KL 散度一阶导数：  
   \[
   \dot{D} = (\pi_k - \mu_k) + \sum_i \pi_i (\mu_i - \pi_i)
   \]
   - **局部项** \((\pi_k - \mu_k)\)：等价于比率方向准则（\(\because \pi_k - \mu_k \propto r-1\)）。  
   - **全局项** \(\sum_i \pi_i (\mu_i - \pi_i)\)：捕捉 softmax 归一化耦合效应，是比率无法观测的部分。  
   - 两者合并后，方向性准则才能准确反映散度实际变化。

2. **截断词汇表下的估计**：生产环境通常只暴露 top-K 概率。  
   - **聚合尾估计器（aggregated-tail）**：将尾部分布合并为一个虚拟 token，直接代入闭式解。  
   - **均匀尾估计器（uniform-tail）**：将尾部质量均匀摊分到 \(n-m\) 个未观测 token 上，尾项除以 \(n-m\) 从而更精确。  
   - 两者计算的 \(\dot{D}\) 差值很小（因尾质量小而 \(n-m\) 很大），实践中行为几乎一致。

3. **掩码规则**（式 13）：  
   \[
   M_t^{\text{pred}} = 
   \begin{cases}
   0, & \text{若 } \text{sign}(\hat{A}_t \dot{D}_t) > 0 \text{ 且 } D_t > \delta, \\
   1, & \text{否则}
   \end{cases}
   \]
   此规则与 DPPO 的框架完全兼容，仅替换方向性准则，不引入新超参数（仅需原有散度阈值 \(\delta\)）。

## 6. 论文的主要结论与发现

- **预测散度掩码优于比率方向准则**：在全部四个设置下，使用预测散度方向（两种估计器）的 DPPO 均比原始 DPPO-TopK-KL 获得更高的训练稳定性和最终准确率；GRPO Clip-Higher 在大模型低精度时崩溃。  
- **散度方向更准确反映散度实际变化**：在 token 级分析中，对于两个准则矛盾的 token，基于散度的方向准则的“不安全保留率”（保留后散度增大）为 34.2%，低于比率准则的 36.9%（降低 2.7 pp）；保留的 token 中散度减小的比例也更高（65.1% vs. 62.9%）。  
- **对阈值不更敏感**：在较紧阈值 \(\delta = 0.05\) 下，预测方向仍改进 DPPO。  
- **聚合尾与均匀尾估计器表现几乎一致**，表明尾部近似稳健。

## 7. 优点

- **方法创新**：首次使方向性准则与散度信任域一致，填补了 DPPO 的缺口；为 softmax 策略推导了一阶导数闭式解，清晰揭示了比率方向遗漏的全局耦合项。  
- **实用落地**：针对 top-K 截断设计了轻量级估计器，几乎无额外计算开销；无需修改底层 RL 框架或引入新超参数。  
- **实验系统**：覆盖多模型规模、多精度，并包含细粒度 token 级分析验证机制，实验设计有说服力。
