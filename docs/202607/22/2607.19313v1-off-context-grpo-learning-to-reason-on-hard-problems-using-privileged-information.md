## 摘要
带可验证奖励的强化学习（RLVR）改进了大型语言模型的推理能力。然而，典型的RLVR方法在困难问题上失败：当模型无法生成任何正确解时，它接收到的学习信号为零。在训练期间提供特权指导（例如解前缀）可以帮助克服这个学习悬崖，通过引导模型走向可获得非零奖励的正确解。我们将这些轨迹称为off-context：它们是从包含特权指导的训练提示中生成的，而目标目标则由没有该指导的原始提示定义。我们引入了Off-Context GRPO（OC-GRPO），这是GRPO的一个最小变体，它使用引导轨迹但应用重要性校正的目标，将更新引导回原始无指导的目标，避免了不校正的引导训练中不稳定的不匹配。实验上，我们的算法在标准数学推理基准上平均比原始GRPO实现了3.9%的绝对改进（13.8%的相对提升），且额外成本可忽略不计。

## Abstract
Reinforcement learning with verifiable rewards (RLVR) improves reasoning in large language models. Yet, typical RLVR approaches fail on difficult problems: when a model cannot generate any correct solutions, it receives \textit{zero} learning signal. Providing privileged guidance during training, such as solution prefixes, can help overcome this learning cliff by steering the model towards {correct solutions with non-zero reward}. {We call these rollouts \textit{off-context}: they are generated from a training prompt that contains privileged guidance, while the target objective is defined by the original prompt without that guidance.} {We introduce} Off-Context GRPO (OC-GRPO), a minimally modified variant of GRPO that uses guided rollouts but applies an importance-corrected objective to steer the update back toward the original unguided objective, avoiding the mismatch that destabilizes uncorrected guided training. Empirically, our algorithm achieves a 3.9\% absolute improvement (13.8\% relative gain) over vanilla GRPO on average across standard mathematical reasoning benchmarks with negligible additional cost.

# Off-Context GRPO 论文总结

## 1. 核心问题与整体含义（研究动机和背景）

**研究动机**：带可验证奖励的强化学习（RLVR）能有效提升大语言模型（LLM）的推理能力，例如 GRPO 算法。然而，在**困难问题**上（模型生成的所有响应均为错误时），奖励全为零，组内优势方差消失，导致梯度为零——即出现 **“学习悬崖”（learning cliff）**。模型无法从这些样本中获取任何有用信号，训练停滞。

**特权信息的作用**：一种常见的应对方案是在训练时提供**特权指导**（如正确解的前缀、提示等），引导模型生成正确响应以恢复非零奖励。但这种做法会引入一个关键问题：训练时的“引导上下文”（g(x)）与推理时的“无引导上下文”（x）不同，导致优化目标不一致，称为 **off-context 问题**。现有方法（如 POPE、BREAD、PrefixRL）虽然使用了引导采样，却未校正这种分布偏移，实际优化的是 `J_guide(θ)` 而非目标 `J(θ)`。本文正是为了解决这一 mismatch。

## 2. 方法论

### 核心思想：Off-Context GRPO (OC-GRPO)

OC-GRPO 是 GRPO 的一种最小修改版本，核心步骤如下：

1. **识别困难问题**：对于给定问题 x，若基础模型 π_ref 在 64 次采样中均无法正确回答，则标记为 hard。
2. **构造引导上下文**：为每个 hard 问题 x 定义 5 级引导水平（通过截取参考解的前缀或生成分层提示），取最小能产生至少一个正确响应的水平，记为 g(x)。（OC-GRPO-Fixed 在训练前用 π_ref 一次性确定引导级别；OC-GRPO-Adaptive 在训练中动态调整。）
3. **采样引导轨迹**：从 π_θ_old(· | g(x)) 中采样 G 条响应 {y_i}，并计算奖励 r(x, y_i) 及组内优势 Â_i。
4. **重要性校正**：使用**每个 token 的重要性权重**替换标准 GRPO 的 ρ_i,t(θ)：
   \[
   \rho^{\text{oc}}_{i,t}(\theta) = \frac{\pi_\theta(y_{i,t} | x, y_{i,<t})}{\pi_{\theta_{\text{old}}}(y_{i,t} | g(x), y_{i,<t})}
   \]
   该权重衡量了在原始上下文 x 下的概率与在引导上下文 g(x) 下的概率之比。
5. **更新策略**：将上述 ρ^oc 代入 GRPO 的 clipped surrogate 目标，保持 KL 正则化项不变，得到 OC-GRPO 损失函数：
   \[
   \mathcal{L}_{\text{OC-GRPO}}(\theta) = \mathbb{E}_{x\sim D}\left[ \frac{1}{G} \sum_{i=1}^G \frac{1}{T_i} \sum_{t=1}^{T_i} \left( \min\left( \rho^{\text{oc}}_{i,t}(\theta) \hat{A}_i, \ \text{clip}(\rho^{\text{oc}}_{i,t}(\theta),1-\epsilon,1+\epsilon)\hat{A}_i \right) - \beta \text{KL}_{i,t}(\theta) \right) \right]
   \]

**关键技术细节**：
- 重要性校正确保目标**无偏地估计原始无引导目标 J(θ)**。
- 理论分析（Lemma 1 和 Theorem 2）表明，该校正会自动对“高度依赖引导的成功轨迹”**打折**（λ+ < 1），对“即使在引导下仍失败的轨迹”**加重惩罚**（λ- > 1），实现**行为感知的信用分配**。
- 相比于直接使用引导目标但不作校正（如 POPE、BREAD），OC-GRPO 避免了训练不稳定（附录 D 展示了无校正时奖励崩塌）。

## 6. 主要结论与发现

1. **OC-GRPO 显著优于基线**：在 Qwen2.5-7B 上，OC-GRPO-Fixed 平均 Pass@1 达 31.7，相对标准 GRPO 提升 13.8%，绝对提升 3.9%；超越最强引导基线 POPE*（+8.7%）。
2. **重要性校正在小模型上至关重要**：在 3B 和 1.5B 规模，未经校正的引导方法（PrefixRL*, BREAD*）性能甚至**低于**原始 GRPO，而 OC-GRPO 仍保持正增益。这表明“后泛化”（back-generalization）依赖于模型容量，而重要性校正机制补足了这一缺陷。
3. **自适应 vs 固定引导**：固定引导（OC-GRPO-Fixed）在 7B 上优于自适应版本（OC-GRPO-Adaptive），且成本更低，因此推荐作为默认方案。
4. **理论支撑**：证明了 OC-GRPO 对 J(θ) 无偏估计，并导出了行为感知的信用分配公式，解释了为何重要校正能有效区分“自主解决”与“依赖提示”的成功。
5. **不同特权信号形式的有效性**：无论使用解前缀还是级联提示，OC-GRPO 均能带来增益，验证了其框架的通用性。

## 8. 不足与局限

- **规模限制**：最大模型为 Qwen2.5-7B，未在更大模型（如 70B 或千亿级）上验证，行为在 frontier 规模可能存在差异。

- **特权信息质量依赖**：主要实验使用参考解前缀（gold prefix），对于部分、噪声或对抗性指导的情况未做研究。级联提示虽然测试，但仍然是高质量信号。
- **无校正失败的实验**：虽然展示了无校正的崩塌，但未详细量化无校正方法在不同难度的衰减曲线。
